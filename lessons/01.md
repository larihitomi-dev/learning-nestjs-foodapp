# Lição 01

## Tema: instalação da aplicação "nestjs" e execução do web sever

É necessário instalar o nodejs, gitbash, estudar documentação do nestjs. A aula resume-se em aprender conceitos básicos sobre dependências de ferramentas e fluxo de estudo prático.

## Objetivo

- Aprender a iniciar um novo projeto nestjs
- Aprender a executar um projeto nestjs
- Aprender conceitos básicos do git
- Aprender modelo básico de estrutura do framework: controllers, models, routing, etc.

## Resultado

- Aprendi a utilizar alguns comandos do git (git init, git commit, git push)
- Aprendi a utilizar o github (criar novo repositório, sincronizar repositórios, configurar chave SSH, etc)
- Aprendi a respeito do uso de um framework (para que serve, como consultar a documentação, como fazer uso, rodar localmente, etc)
