# Lição 03

## Tema: Adição de filtro de categorias

Vamos adicionar de maneira básica o atributo "categoria" no objeto de restaurantes, com finalidade de criar filtro de restaurantes e pratos por suas categorias.

## Objetivo

- Adicionar nova propriedade "categoria" ao objeto Restaurantes
- Criar rota para exibir as categorias existentes (sem duplicidade)
- Criar rota para filtrar pratos pela categoria
- Estudar funções do javascript para filtrar restaurantes ou pratos por busca selecionada pelo usuário
- Reforçar aprendizado de criação e uso de rotas
- Estudar ferramentas complementares para o uso desta aplicação por meio de requisições HTTP. Ex.: postman

## Resultado

- Aprendi a adicionar um novo atributo dentro de uma classe já existente
- Aprendi a criar uma nova rota
- Aprendia a utilizar o método construtor
- Aprendi a pegar um atributo dentro de um objeto
- Aprendi a remover duplicidades dentro de um array
