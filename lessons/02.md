# Lição 02

## Tema: Desenvolvimento inicial com fim didático de plataforma de food delivery

Nesta lição será iniciado o escopo básico de uma plataforma de pedidos de refeições, também adoção de componentes do framework a fim de criar um sistema de roteamento básico para: /restaurantes e /comidas.

## Objetivo

- Aprender a criar novas rotas no projeto
- Aprender conceitos básicos sobre organização de projetos
- Aprender aplicabilidade de POO em projeto nestjs
- Criar familiaridade com linguagem typescript

## Resultado

- Aprendi a criar rotas e usá-las via postman
- Aprendi a importar classes entre os arquivos
- Aprendi mais a respeito de models, sobre: como criá-las, usá-las, etc.
- Aprendi a interagir com objetos
- Aprendi sobre tipagem de models, atributos, propiedades, etc.
- Reforcei alguns fundamentos com git
